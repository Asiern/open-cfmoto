# Packages Overview

This section documents the main monorepo packages and their responsibilities.

## Workspace map

- `apps/mobile`: React Native app using Expo.
- `packages/ble-protocol`: BLE protocol codec, command builders, routing, auth flow.
- `packages/cloud-client`: Cloud API client for app backends.
- `tools/apk-analysis/findings`: Decompiled APK findings and protocol evidence.

## Navigation tips

- Use **Findings** for reverse engineering evidence.
- Use **Packages** for implementation and API behavior.
- Use **Project** for architecture, audits, and validation notes.
